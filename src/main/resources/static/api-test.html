📋 任务管理 API /api/v1/tasks
POST /api/v1/tasks
创建任务（自动触发调度）
GET /api/v1/tasks
获取任务列表（支持筛选）
GET /api/v1/tasks/{taskId}
获取任务详情
PATCH /api/v1/tasks/{taskId}/status
更新任务状态（SUCCESS/FAILED 会自动写日志）
🤖 机器人管理 API /api
GET /api/robots
获取机器人列表
GET /api/robots/pose
获取机器人实时位姿（由 rosbridge /amcl_pose 自动同步）
POST /api/robot/goal
设置机器人目标点（内存 + 简化直线路径）
POST /api/robots/pose
更新机器人实时位姿（前端上报坐标到后端）
GET /api/robot/path?robotId=xxx
获取规划路径（简化直线插值）
📜 日志查询 API /api/v1/logs
GET /api/v1/logs
获取日志列表（支持 type / referenceId 筛选）
🗺️ SLAM 地图与路径规划 API /api/v1/scheduler/slam
GET /api/v1/scheduler/slam/map
获取当前激活地图数据（OccupancyGrid 格式 + 障碍物）
GET /api/v1/scheduler/slam/maps
获取地图列表（元数据，不含二进制）
POST /api/v1/scheduler/slam/maps/{mapId}/switch
切换激活地图
POST /api/v1/scheduler/slam/map/reset
重置地图（从数据库重新加载原始数据，清空障碍物）
GET /api/v1/scheduler/slam/obstacles
获取障碍物列表
POST /api/v1/scheduler/slam/obstacles
添加障碍物 / 空气墙
POST /api/v1/scheduler/slam/path/plan
A* 路径规划（基于当前栅格地图 + 障碍物）
🧠 LLM 自然语言解析 API /api/v1/scheduler/llm
POST /api/v1/scheduler/llm/parse
解析自然语言指令，自动拆分为子任务并存入数据库
📡 ROS2 通信 API /api/v1/scheduler/ros
GET /api/v1/scheduler/ros/status
查询 rosbridge 连接状态与消息计数
POST /api/v1/scheduler/ros/goal
发送导航目标到 ROS2（发布 /goal_pose）
